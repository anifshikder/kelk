The Digital Contagion Network

By Khaoula El Khalil

Far-right attackers are routinely called lone wolves. Their own manifestos say otherwise. This map traces the documented chain of citation between major attacks: who studied, named, and set out to outscore whom. Hover an attack to send the trace backward through everyone it drew from. What looks like isolated violence is a transmission network with an index case.

Reading the map. The horizontal axis is time. An edge is drawn only where a perpetrator explicitly named or reproduced an earlier attacker in a manifesto, forum post, or court record. The trace follows those edges backward — and keeps going, because sources had sources, terminating at the two pre-Christchurch control cases (2011 and 2015).

Scope. The eight solid nodes are the corpus documented in the dissertation's annex table (Christchurch, Poway, El Paso, Bærum, Halle, Hanau, Buffalo, Bratislava). The two dashed nodes are the pre-Christchurch control cases (Breivik 2011, Roof 2015) against which the later gamified pattern is measured. Hanau appears without citation links: it is retained as a border case where no explicit reference to an earlier attacker is documented.

Method & restraint. Nodes are labelled by place and date rather than name by default, following the "No Notoriety" principle these attackers were actively courting. Casualty figures appear as historical record, never as points. No manifesto text, imagery, or slogans are reproduced; only the fact and source of each citation.
